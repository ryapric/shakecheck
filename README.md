Shakecheck: [`shellcheck`](https://github.com/koalaman/shellcheck) for Makefile
shell code
================================================================================

Does what it says above.

Makefile MUST define a `SHELL` variable, just like `shellcheck` requires a
shebang.

Invoke via:

    ./shakecheck.sh <Makefile>
